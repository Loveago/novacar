Build a modern gift card buy-back marketplace web app where users can browse gift cards like a marketplace, submit cards to sell, and an admin manages gift card listings, images, and prices.
NO Mistakes 

Tech Stack

Frontend: Next.js (App Router)

Backend: Node.js using Next.js API routes

Database: PostgreSQL with Prisma ORM

Authentication: NextAuth Credentials Provider

Styling: Tailwind CSS + modern UI components

File Storage: Local disk storage (server file system)

Core Concept

This is a gift card marketplace UI where:

Users see gift cards displayed as product cards (logo, name, rate, price info).

Users select a gift card and submit details to sell it.

Admin manages gift cards like a marketplace catalog (add/edit/delete with images and rates).

Admin reviews submissions and manually sends payment.

User Roles

Register and log in

Full CRUD for gift cards (name, image, rate, description)

Edit prices and exchange rates

View and manage submissions

Approve or reject submissions

Mark transactions as paid
