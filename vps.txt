# Deploying Joy Exchange Marketplace on a VPS (clean, safe, copy-paste)

These steps assume:
- Ubuntu 22.04/24.04.
- SSH access.
- Domain: **joy-exchange.com** (Cloudflare or any DNS provider).
- Repo path: `/var/www/novacar/giftcard-marketplace`.

---
## 0. (Optional) Add 1GB swap for low-memory VPS
```bash
sudo swapoff -a
sudo rm -f /swapfile
sudo dd if=/dev/zero of=/swapfile bs=1M count=1024 status=progress
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile
echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab
```

---
## 1. Prepare the VPS (official packages only)
```bash
sudo apt update && sudo apt upgrade -y
sudo apt install -y git curl ca-certificates gnupg build-essential nginx ufw
```

---
## 2. Install Node.js LTS (official NodeSource)
```bash
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt install -y nodejs
node -v && npm -v
```

---
## 3. Install PM2
```bash
sudo npm install -g pm2
```

---
## 4. Install and configure PostgreSQL (local DB)
```bash
sudo apt install -y postgresql postgresql-contrib
sudo systemctl enable --now postgresql
```

Create a database + user (change password):
```bash
sudo -u postgres psql <<'SQL'
CREATE USER joyexchange WITH PASSWORD 'Loverboy888';
CREATE DATABASE joyexchange OWNER joyexchange;
ALTER USER joyexchange CREATEDB;
SQL
```

---
## 5. Clone the repo
```bash
sudo mkdir -p /var/www/novacar
sudo chown -R $USER:$USER /var/www/novacar
cd /var/www/novacar
git clone https://github.com/Loveago/novacar.git giftcard-marketplace
cd /var/www/novacar/giftcard-marketplace
```

---
## 6. Create `.env` (in /var/www/novacar/giftcard-marketplace)
Generate secrets (optional):
```bash
openssl rand -hex 32
```

Create the file:
```bash
cat <<'EOF' > .env
DATABASE_URL="postgresql://joyexchange:Loverboy888@localhost:5432/joyexchange"
AUTH_SECRET="your-long-random-strinssafasfvwef3r3rg"
CARD_SECRET="your-other-long-strr3r23er23r2r23ring"
NEXTAUTH_URL="https://joy-exchange.com"
RESEND_API_KEY="re_cPRPAQgm_EuYBEhUpJKVemz2PcuSWAhhZ"
EMAIL_FROM='Joy Exchange <noreply@joy-exchange.com>'
AUTO_SEED="true"
SEED_ADMIN_EMAIL="admin@joy-exchange.com"
SEED_ADMIN_PASSWORD="Eliteguy1122@"
EOF
```

---
## 7. Install dependencies and build
```bash
npm insta
cd /var/www/novacar/giftcard-marketplace/gitcard-marketplacell
npx prisma migrate deploy
npx prisma generate
node prisma/seed.cjs   # optional one-time seed
npm run build
```

If you used AUTO_SEED, set it back to false after first run:
```bash
perl -0pi -e 's/AUTO_SEED="true"/AUTO_SEED="false"/g' .env
```

---
## 8. Start with PM2
```bash
cd /var/www/novacar/giftcard-marketplace
pm2 start npm --name joy-exchange -- start
pm2 save
pm2 startup
```

---
## 9. Configure Nginx reverse proxy
```bash
sudo tee /etc/nginx/sites-available/joy-exchange <<'EOF'
server {
  listen 80;
  server_name joy-exchange.com www.joy-exchange.com;

  location / {
    proxy_pass http://127.0.0.1:3000;
    proxy_http_version 1.1;
    proxy_set_header Upgrade $http_upgrade;
    proxy_set_header Connection 'upgrade';
    proxy_set_header Host $host;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_set_header X-Forwarded-Proto $scheme;
    proxy_cache_bypass $http_upgrade;
  }
}
EOF

sudo ln -s /etc/nginx/sites-available/joy-exchange /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

---
## 10. DNS (Cloudflare or any DNS)
- `@` A record → VPS IP
- `www` CNAME → `joy-exchange.com`

---
## 11. HTTPS (Certbot)
```bash
sudo apt install -y certbot python3-certbot-nginx
sudo certbot --nginx -d joy-exchange.com -d www.joy-exchange.com
```

---
## 12. Firewall (recommended)
```bash
sudo ufw allow OpenSSH
sudo ufw allow 80
sudo ufw allow 443
sudo ufw enable
sudo ufw status
```

---
## 13. Deploy updates
```bash
cd /var/www/novacar/giftcard-marketplace
git pull origin main
npm install
npx prisma migrate deploy
npm run build
pm2 restart joy-exchange
```

---
## 14. Troubleshooting
- **App not reachable**: `pm2 logs joy-exchange` and `sudo journalctl -u nginx`.
- **Env vars missing**: edit `.env`, then `pm2 restart joy-exchange --update-env`.
- **DB errors**: ensure PostgreSQL is running: `sudo systemctl status postgresql`.

That’s it. This guide uses only official Ubuntu packages + NodeSource LTS to avoid outdated or random package sources.
